# scrabble_score

Very simply, this script counts the scrabble score based on a string input.

## Examples

```
$ python3 score.py Matthew Jorgensen
Scrabble score: 32
```

```
$ python3 score.py
What is your name? Matthew Jorgensen
Scrabble score: 32
What is your name?
...
```
